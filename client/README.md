🎨 Tailwind Default Color Usage Guide

| UI Element           | Recommended Tailwind Class                     | Why / When to Use                                 |
| -------------------- | ---------------------------------------------- | ------------------------------------------------- |
| **Primary Button**   | `bg-indigo-600 hover:bg-indigo-700 text-white` | Primary action like "Login", "Register", "Submit" |
| **Secondary Button** | `bg-cyan-500 hover:bg-cyan-600 text-white`     | Secondary actions like "Edit", "More Info"        |
| **Accent Elements**  | `text-emerald-500` or `bg-emerald-100`         | Highlights, success icons, or badge elements      |
| **Danger Button**    | `bg-red-600 hover:bg-red-700 text-white`       | Destructive actions like "Delete", "Logout"       |
| **Warning Message**  | `bg-yellow-100 text-yellow-800`                | Warnings, validation alerts                       |
| **Success Message**  | `bg-green-100 text-green-800`                  | Confirmation messages like "Profile updated"      |
| **Card Background**  | `bg-white` or `bg-gray-50`                     | For dashboards, forms, profile sections           |
| **Text (Primary)**   | `text-gray-900`                                | Main content or headlines                         |
| **Text (Secondary)** | `text-gray-600` or `text-gray-500`             | Descriptions, subtitles, less important text      |
| **Form Inputs**      | `border-gray-300 focus:ring-indigo-500`        | Clean inputs with focus indication                |
| **Page Background**  | `bg-gray-50` or `bg-gray-100`                  | Soft background color                             |
| **Navbar**           | `bg-white shadow text-gray-800`                | Clean and readable navigation bar                 |
